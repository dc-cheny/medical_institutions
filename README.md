# medical_institutions
**该资源包含全国医疗机构列表**

## Requirements
numpy  
tqdm  
pandas  
requests

```{bash}
pip install -U -r requirements.txt # 输入该指令一键安装
```

## Usage
```{bash}
python main.py # 获取数据
```

```{bash}
python combine_data.py # 数据处理及合并
```

## Source
http://www.pharmnet.com.cn/search/template/yljg_index.htm
